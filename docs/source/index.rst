Installation
------------

.. toctree::
    :maxdepth: 2
    :hidden:

    installation

Installation should be as easy as executing this command in your chosen terminal::

    $ pip install youtube_feeder

The source code for this project is `hosted on Github <https://github.com/scolby33/youtube_feeder>`_.
Downloading and installing from source goes like this::

    $ git clone https://github.com/scolby33/youtube_feeder
    $ cd youtube_feeder
    $ pip install .

If you intend to install in a virtual environment, activate it before running :code:`pip install`.

See :ref:`installation` for further information about installing :code:`youtube_feeder` in all manner of ways.
